# Syntax - Cambly's design system

# Setup

## Install nvm

```bash
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.3/install.sh | bash
```

## Install node

```bash
nvm install
```

## Install dependencies

```bash
npm install -g pnpm
pnpm install
```

## Run the dev server

```bash
pnpm dev
```
