# Syntax - Cambly's design system

## Setup

### Install nvm

```bash
# Install nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.3/install.sh | bash
# Install node
nvm install
# Install dependencies
npm install -g pnpm
pnpm install
# Run the dev server
pnpm dev
```
